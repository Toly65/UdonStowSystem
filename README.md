# Toly's Udon Stow System

Tired of holding things in your hand just to take them somewhere?

This package adds an H3VR-style, on-person stow point system for VRChat worlds built with Udon/UdonSharp. Players can attach supported items to body-mounted stow points (acting as virtual pockets, pouches, or holsters), keep their hands free, and quickly pull items back out when needed. It can also serve as a foundation for a simple, diegetic inventory system.

## What It Does

The Udon Stow System lets players:

- Stow compatible items onto attachment points on their character — think hip holsters, back slots, belt pouches, or virtual pockets.
- Free up their hands while moving around, climbing, interacting, or swapping tools.
- Retrieve stowed items quickly instead of hunting for where they were dropped.
- Organize loadout-style gameplay with intentional item placement.
- Use stow points as a basic inventory system for carrying keys, ammo, currency, or other small props.

## Core Features

- H3VR-inspired on-person stow behavior.
- Attachment point workflow designed for VR interaction.
- Supports "put away now, use later" gameplay loops.
- Functions as virtual pockets or pouches — ideal for carrying small items without a traditional UI inventory.
- Great for utility items, weapons, tools, props, and inventory-lite systems.
- Works well for worlds that want physical item management without full menu inventory.
- Can be extended into a basic inventory system by treating stow points as categorized storage slots (e.g. ammo pouch, key pocket, tool belt).

## Typical Use Cases

- Combat/arena worlds where players need both mobility and access to gear.
- Exploration or roleplay worlds where carrying multiple props feels natural.
- Utility-focused maps where users rotate between tools often.
- Adventure or puzzle worlds where players collect keys, items, or clues — using stow points as pockets or a lightweight inventory.
- RP settings where characters carry distinct items on their person (e.g. a coin purse on the belt, a map in a back pocket, a dagger in a boot sheath).

## Why Use This Instead of Hand-Carrying?

Keeping an item physically in hand the whole time can be awkward and limiting.
With stow points, players can move naturally, interact with the world, and still keep important items on their person for fast access — just like using real pockets, a belt, or a backpack.

## Notes

- This system is intended for creators who want physical, diegetic item storage.
- Setup details may vary depending on your item prefabs and interaction scripts.
